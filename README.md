# 🚀 Scalable Web App with NLB and Auto Scaling

## 🎯 Project Overview

This project demonstrates how to deploy a highly available and scalable web application on AWS using Amazon EC2, Network Load Balancer (NLB), and Auto Scaling Group.

The architecture efficiently distributes network traffic across multiple EC2 instances while automatically scaling resources based on traffic demand.

---

## 🛠 AWS Services Used

* Amazon EC2
* Network Load Balancer (NLB)
* Auto Scaling Group (ASG)
* Security Groups
* Target Group
* Amazon VPC
* Internet Gateway

---

## 🏗️ Architecture

The application is deployed on multiple EC2 instances behind a Network Load Balancer. Auto Scaling automatically launches or terminates EC2 instances based on incoming traffic.

---

## 🔄 Workflow

1. User sends a request to the Network Load Balancer.
2. NLB forwards the request to healthy EC2 instances.
3. Auto Scaling monitors resource utilization.
4. New EC2 instances are launched during high traffic.
5. Unused instances are terminated during low traffic.
